Was sind APIs?

API steht für "Application Programming Interface" und lässt sich als Programmierschnittstelle übersetzen.
Eine API ist ein Satz von Befehlen, Funktionen, Protokollen und Objekten, die Programmierer verwenden können, um eine Software zu erstellen oder mit einem externen System zu interagieren.
Beim Programmieren vereinheitlichen APIs die Datenübergabe zwischen Programmteilen, etwa Modulen, und Programmen.
Die Schnittstellen dienen quasi als Zugangspunkt für eine andere Software.
